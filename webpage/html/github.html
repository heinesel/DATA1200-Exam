    Git and GitHub

    Our group collaborated using GitHub and Git which are useful tools for our project. Git is an Open-source distributed version control system that can allow multiple people to work together on a project at the same time without interfering with each others work, (Robert, 2023). In other words, you can merge your code with somebody else easily.

    GitHub is a cloud-based collaboration platform where developers can create, store, manage and share their code, (GitHub Docs, 2024). This platform is made possible by the software Git, and that is how they are connected.

    Collaboration

    The group members committed to the main branch multiple times and each change added up in the commit history where we had track of what was changed and by who. It was easier to recognize the changes by seeing the difference in the previous and newer version of the code, as shown in the examples above in the images regarding commit history.

    By looking into this even more, you can see specific changes made as shown in the images such as changing in font-sizes, adding/removal of element-styling and adding additional html-comments to communicate in the best possible way with others. It helps with preventing more errors or any form of confusion.
